Repositório - Protótio de site para lojas

Descrição

Esse repositório apresenta um programa elaborado com o uso de html e css para representar um site para uma loja de utilidades
Tal solução é referente ao exercício 5 da matéria de "Desenvolvimento de Interfaces Web", ministrado pelo professor Edwaldo Soares Rodrigues, do curso de Sistemas de Informação da PUC-MG e foi realisado durante o primeiro período do referido curso.
